指南

如何使用这个书签管理器

快速开始

  • 用搜索查找标题、网址、域名、标签或分类。
  • 在侧边栏按分类、标签、域名和链接状态筛选。
  • 列表区会显示当前结果数量，并支持一键清除筛选。

书签操作

  • 点击卡片或“访问”即可打开原始网站。
  • 打开“详情”查看网址、来源文件夹、分类、标签和描述。
  • 在网格与列表视图之间切换，调整浏览密度。

整理与导出

  • 报告汇总统计、重复、死链和建议。
  • 可导出调整后的 JSON、Markdown 总览，或浏览器可导入的 HTML。
  • JSON 导出会包含保存在当前浏览器中的暂存调整。

本地暂存

  • “详情”里的高级编辑只保存在当前浏览器中。
  • 它们不会直接修改源 bookmarks.json 或浏览器书签。
  • 要保留编辑，请导出 JSON 并主动替换源数据。
